# smart-csv-excel-visualizer
An embeddable tool that turns `.csv` or `.xlsx` files into interactive charts and dashboards — 100% client-side.

## Features
- Drag & drop CSV/XLSX or load from URL
- Auto-detects column types → suggests charts
- Supports Sum, Average, Count, Median
- Export chart as PNG
- Shareable hash links (no backend)
- Works offline, no server needed
